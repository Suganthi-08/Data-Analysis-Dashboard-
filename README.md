# Data-Analysis-Dashboard-
A Data Analysis Dashboard is an interactive visual interface used to monitor, analyze, and interpret data in real time. It consolidates large volumes of raw data into meaningful insights through charts, graphs, KPIs, and visual indicators, enabling users to make informed decisions quickly.

# Samsung-Sales-Data-Analysis (Interactive Dashboard Creation using Power BI)

## Project Objective
The objective of this project is to analyze Samsung product sales data and create an interactive dashboard to monitor business performance.  
This dashboard helps analyze revenue trends, product category performance, regional sales distribution, and yearly growth patterns.

The dashboard enables businesses to understand their sales performance and make better data-driven decisions.

## Dataset used
- <a href="https://github.com/yourusername/Samsung-Sales-Dashboard/blob/main/Dataset/samsung_sales_data.xlsx">Samsung Sales Data</a>

## Questions (KPIs)
- What is the **total revenue generated**?
- How many **units were sold**?
- Which **product category generates the highest revenue**?
- What are the **maximum and minimum sales values**?
- How does **sales trend change across years**?
- Which **region contributes the most revenue**?

## Process
- Verify data for missing values and anomalies.
- Perform data cleaning and transformation.
- Create data model for analysis.
- Develop measures using calculations.
- Build interactive visuals and KPIs.
- Add filters for year, category, and region.
- Design an interactive dashboard layout.

## Dashboard Features
- Total Revenue KPI
- Units Sold KPI
- Maximum and Minimum Sales
- Revenue by Category (Bar Chart)
- Yearly Sales Trends (Line Chart)
- Revenue by Region (Donut Chart)
- Detailed Sales Table
- Interactive Filters (Year, Category, Region)

## Project Insights
- Asia region contributes the highest share of sales revenue.
- Smart TV and Galaxy series generate the largest revenue.
- Sales show growth from 2021 to 2023 with a slight decline in 2024.
- Regional sales distribution highlights strong market presence in Asia.

## Conclusion
This dashboard provides a clear overview of Samsung sales performance across different product categories and regions.  
It helps businesses track key performance indicators and identify opportunities for growth using data-driven insights.
